# Molecular_Translation

### 1. Data File Generation
```powershell
python data_generator.py [CID-SMILES file name] [Data Size]
```



**CID-SMILES File Source**
https://ftp.ncbi.nlm.nih.gov/pubchem/Compound/Extras/CID-SID.gz 



**Example**

```powershell
python data_generator.py CID-SMILES-TEST 100
```
